# app_design

페르소나를 선정 후, 사용자 시나리오를 구상하여 어플의 정보를 설계했습니다.<br>
앱 페이지를 디자인 하기 전에 와이어프레임을 스케치한 후<br>
UX를 위해 카카오Oven을 사용하여 제작한 프로토타입을 토대로 워크플로우를 구상하고 앱페이지를 디자인했습니다.<br>
